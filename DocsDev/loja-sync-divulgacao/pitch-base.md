# Pitch Base — LojaSync Divulgação

## Proposta central

**LojaSync automatiza o cadastro de produtos no Byte Empresa a partir do romaneio ou nota fiscal.**

Quem usa para de digitar produto por produto e passa a importar tudo de uma vez — com grade, preço e código corretos.

---

## Respostas para objeções comuns

| Objeção | Resposta |
|---|---|
| "Parece complicado" | "É mais simples que o próprio ByteEmpresa — você só manda o arquivo e o sistema faz o resto." |
| "Não tenho tempo agora" | "Leva 2 minutos ver o vídeo. Se não fizer sentido, sem compromisso nenhum." |
| "Já tenho alguém que faz isso" | "Entendo. O LojaSync não substitui a pessoa — ajuda ela a fazer em minutos o que hoje leva horas." |
| "Quanto custa?" | "Trabalhamos com planos acessíveis para pequenas lojas. Posso te mandar os detalhes depois de mostrar como funciona?" |
| "Tenho medo de perder dados" | "O sistema não apaga nada do ByteEmpresa — ele só adiciona os produtos que você mandar. Pode revisar antes de confirmar." |

## Call to action padrão

- **Primário:** "Posso mandar um vídeo de 2 minutos mostrando como funciona?"
- **Secundário:** "Quando seria um bom momento para uma demonstração rápida?"
- **Fallback:** "Mando o link para você ver no seu tempo, sem compromisso."

---

## O que NÃO fazer no pitch

- Não listar funcionalidades técnicas antes de confirmar a dor
- Não mencionar "inteligência artificial" ou "machine learning" — gera desconfiança
- Não pedir reunião longa no primeiro contato
- Não mandar mais de uma mensagem sem resposta no mesmo dia
- Não mencionar concorrentes pelo nome
